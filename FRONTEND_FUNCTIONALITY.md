## 🚀 使用方法

1. **启动服务器**
   ```bash
   php artisan serve
   ```

2. **访问首页**
   - 打开 `http://localhost:8000`

3. **注册/登录**
   - 点击 "Sign In" 或访问 `/login`
   - 注册新账户或使用已有账户登录

4. **使用功能**
   - 登录后可以访问所有功能模块
   - 所有操作都会实时保存到数据库

## 📝 注意事项

- 所有 API 调用都需要认证（除了登录和注册）
- Token 存储在 localStorage 中
- 如果 token 过期，会自动跳转到登录页
- 所有操作都有成功/失败提示

## 🐛 故障排除

如果功能不工作，检查：
1. 服务器是否运行 (`php artisan serve`)
2. 数据库是否连接正常
3. 浏览器控制台是否有错误
4. Token 是否有效（尝试重新登录）
